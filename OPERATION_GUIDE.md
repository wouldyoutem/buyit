# 📹 우주꿀템 운영 가이드

> 유튜브 쇼츠 → 사이트 유입 → 쿠팡 구매 최적화 시스템

## 🎯 운영 플로우

```
1. 쿠팡 파트너스 상품 선정
2. 유튜브 쇼츠 제작 & 업로드
3. 사이트에 상품 추가 (이 문서 참고)
4. 쇼츠 설명란에 사이트 링크 추가
```

## 🤖 자동 검증 시스템

**profiles.json 업데이트 시 자동으로:**
- ✅ 필수 필드 검증
- ✅ OPERATION_GUIDE.md 준수 여부 확인
- ✅ 유튜브 쇼츠 최적화 팁 자동 생성!

### 로컬에서 검증하기

상품 추가 전에 미리 확인하세요!

```bash
npm run check-product
```

또는

```bash
npm run add-product
```

**자동으로 제공되는 정보:**
- 📝 추천 제목 3가지
- 📄 유튜브 설명란 템플릿
- 🏷️ 추천 해시태그
- 🎥 촬영 팁
- ⏰ 업로드 최적 시간

## 📝 새 상품 추가하는 법

### 1단계: 정보 수집
- ✅ 쿠팡 파트너스 링크
- ✅ 유튜브 쇼츠 URL (나중에 추가 가능)
- ✅ 제품명
- ✅ 내가 느낀 핵심 포인트

### 2단계: profiles.json 파일 수정

파일 위치: `public/profiles/profiles.json`

**새 상품을 맨 위에 추가하세요!** (최신순으로 표시됨)

```json
{
  "products": [
    {
      "id": "7",
      "productName": "제품명 (예: 다이슨 V15 디텍트 무선청소기)",
      "category": "생활가전",
      "description": "핵심 설명 (간결하게, 왜 좋은지)",
      "image": "https://github.com/user-attachments/assets/20f2b4e1-07a3-4f83-9b1a-6b48d5f2298f",
      "coupangLink": "쿠팡 파트너스 링크 전체 URL",
      "youtubeShorts": "https://www.youtube.com/shorts/XXXXXXX",
      "features": [
        "특징 1",
        "특징 2", 
        "특징 3"
      ],
      "highlight": "한 줄 핵심 포인트 (예: 소음 적고 흡입력 최고)",
      "dateAdded": "2025-01-20"
    },
    // 기존 상품들...
  ]
}
```

### 📌 중요! 가격/평점/리뷰수는 입력하지 마세요
- ❌ 가격: 실시간 변동됨
- ❌ 평점: 계속 바뀜
- ❌ 리뷰수: 계속 늘어남
- ✅ **"쿠팡 최저가 확인하기" 버튼으로 직접 확인하도록 유도!**

### 3단계: 로컬 검증 (선택사항이지만 권장!)

```bash
npm run check-product
```

**자동으로 확인해주는 것:**
- 필수 필드 누락 여부
- 금지된 필드 사용 여부 (가격, 평점 등)
- features 개수 (3개 권장)
- 유튜브 쇼츠 최적화 팁!

### 4단계: 커밋 & 푸시

```bash
git add public/profiles/profiles.json
git commit -m "➕ 신상품 추가: 제품명"
git push origin main
```

**GitHub Actions가 자동으로:**
- 검증 실행
- 유튜브 최적화 팁 로그 출력
- 배포 (약 2-3분 소요)

## 🎬 유튜브 쇼츠 최적화 팁

### 자동 생성 시스템 활용!

상품 추가 후 `npm run check-product` 실행하면:
- 해당 상품에 맞는 제목 3가지 제안
- 설명란 템플릿 자동 생성
- 해시태그 추천
- 촬영 팁 제공

### 수동 템플릿 (참고용)

#### 제목 예시
```

✅ "가성비 최강 [제품명] 리뷰"
```

#### 설명란 템플릿
```
🛒 최저가 구매 링크: https://wouldyoutem.github.io/buyit/

📌 타임라인
0:00 개봉
0:15 주요 기능
0:35 실사용 후기
0:50 총평

💬 댓글로 궁금한 점 남겨주세요!

#꿀템 #가성비 #쿠팡추천 #[제품명]

※ 쿠팡 파트너스 활동으로 일정 수수료를 받습니다
```

## 🎯 상품 선정 체크리스트

### 필수 요소
- [ ] 내가 직접 써본 제품 (또는 확신하는 제품)
- [ ] 쿠팡 파트너스 링크 생성 가능
- [ ] 쇼츠로 만들 매력 포인트 있음
- [ ] 사람들이 관심 가질만한 제품

### 제품명 작성법
```
❌ 나쁨: "청소기"
✅ 좋음: "다이슨 V15 디텍트 무선청소기"
```

### 설명 작성법 (본인 말투로!)
```
❌ 나쁨: "좋은 청소기입니다"
✅ 좋음: "레이저로 먼지를 감지하는 똑똑한 청소기! 150분 장시간 사용 가능"
```

### 하이라이트 작성법 (핵심 한 줄)
```
❌ 나쁨: "좋아요"
✅ 좋음: "소음 적고 흡입력 최고"
✅ 좋음: "가족 많으면 이거 최고"
✅ 좋음: "재택근무하면서 걷기 최고"
```

## 🔧 문제 해결

### Q: 사이트에 상품이 안 보여요
A: GitHub Actions 배포 완료까지 2-3분 대기 → 새로고침

### Q: JSON 문법 오류
A: [JSON Validator](https://jsonlint.com/)로 검증
   - 쉼표 위치 확인
   - 따옴표 짝 확인
   - 마지막 항목 뒤 쉼표 제거

### Q: 검증 스크립트가 실행 안 돼요
A: Node.js 설치 확인 → `node --version`

### Q: 이미지를 바꾸고 싶어요
A: 일단은 기본 이미지 사용 (나중에 제품별 이미지 추가 가능)

## 🚀 프로 팁

### 상품 선정 기준
- 직접 써보고 좋았던 제품
- 쇼츠에서 보여줄 게 많은 제품
- 사람들이 궁금해할만한 제품
- 시즌/트렌드에 맞는 제품

### 업로드 타이밍
- **최적**: 저녁 7-9시
- **차선**: 점심 12-1시
- **주말**: 오전 10시, 오후 3시

### 꾸준함이 핵심!
- 일주일에 3-5개 쇼츠 목표
- 꾸준히 올리면 알고리즘이 밀어줌
- 댓글 소통도 중요!
